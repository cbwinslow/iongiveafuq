## BATCH GENERATION SETTINGS

### For Consistent Character Generation

**Recommended AI Settings:**
- **Style Consistency:** Use "character reference" or "style reference" features
- **Quality:** Maximum quality settings (8K, --quality 2 in Midjourney)
- **Aspect Ratios:** 
  - Portraits: 2:3 or 4:5
  - Full body: 3:4
  - Environment: 16:9
  - Merchandise: 1:1
- **Negative Prompts:** "anime style, cute, cheerful, clean, pristine, family-friendly"

### Video Generation Best Practices
- **Length:** 5-15 seconds for character loops
- **Frame Rate:** 24fps minimum for smooth animation
- **Resolution:** 1920x1080 minimum
- **File Format:** MP4 with H.264 codec
- **Loop Points:** Ensure first and last frames match for seamless loops
